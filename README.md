# Roles

## Perquisites

- [Python](https://www.python.org/downloads/)
- [PDM](https://pdm.fming.dev/)

## Steps

```bash
❯ pdm venv create                  # Create a virtual environment
❯ pdm use                          # Set the virtual environment
❯ source $(pdm venv activate)      # Activate the virtual environment (bash)
❯ pdm run install_local            # Install the local packages 
❯ pdm run start                    # Run the application
```
